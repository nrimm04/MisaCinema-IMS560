# MisaCinema - Advanced Web Application & NoSQL DBMS

🌍 **Live Demo:** [MisaCinema Web App](https://misacinemaa.up.railway.app)

MisaCinema is a dynamic, responsive web platform designed for digital movie browsing and management. Developed as part of the **IMS560: Advanced Database Management System** coursework, this project explores full-stack engineering without relying on heavy traditional monolithic frameworks, focusing heavily on optimized NoSQL database configurations.

---

## 🚀 Key Features
* **Dynamic Movie Browsing:** Custom-engineered frontend user interface allowing users to seamlessly browse through rich movie catalogs.
* **NoSQL Schema Optimization:** Built a robust data layer completely from scratch using non-relational database architecture.
* **Low-Latency Handling:** Tailored queries to ensure fast document retrieval and low-latency rendering of content.
* **Admin Management:** Secured database pathways for updating, deleting, and inserting new movie data documents.

## 🛠️ Tech Stack & Tools
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
* **Backend:** Native PHP (Core Logic)
* **Database:** MongoDB (Cloud Cluster via MongoDB Atlas)
* **Deployment & Hosting:** Railway
* **GUI Management:** Studio 3T
* **Local Server Environment:** Laragon

## 📁 Database Architecture
Unlike traditional relational models, the database backend utilizes a document-oriented structure inside MongoDB. High-performance index profiling was configured through **Studio 3T** to tackle complex aggregation pipelines and handle scalable, unstructured datasets with high efficiency.
